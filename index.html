Decode IS 2502 Like a Pro

College theory is over. Ye practical guide aapko sikhayegi ki site par sariya (rebar) ka bending, fixing aur BBS (Bar Bending Schedule) kaise kaam karta hai.

Code Ka Purpose Kya Hai?

IS 2502 rules set karta hai ki concrete me daalne se pehle steel bars ko kaise bend aur fix karna hai taaki structural integrity maintain rahe aur steel fail na ho.

Tolerances (Allowed Error)

Site pe 100% accuracy mushkil hai. IS code allow karta hai:
👉 Length > 1m : +50mm / -25mm
👉 Bending dimensions : +10mm / -10mm

Kyu Zaroori Hai?

Galat bending se micro-cracks aate hain steel me. Agar standard hook length ya bend radius maintain nahi kiya, toh earthquake ke time steel concrete se slip ho jayega.

Bend Deduction Rule

Bhai, jab hum steel bar ko modte hain (bend karte hain), toh outer zone me tension ki wajah se bar thoda **lamba (elongate)** ho jata hai. Agar hum ye badhi hui length minus nahi karenge, toh site pe bar lamba kaat jayega. Ise Bend Deduction kehte hain.

Formula: 1 × d (Where d = diameter of bar)
